发现 /usr/bin/gdb 可以使用

GDB 提权

提权命令解析：

  • nx：不加载任何 .gdbinit 初始化文件，防止被干扰

  • ex：启动时立即执行后面的命令

  • import os; os.setuid(0)：调用 Linux 底层的 setuid() 系统调用，将当前进程的真实用户 ID 强制修改为 0

  • !bash：启动一个新的 bash 终端

  • -ex quit：启动 bash 后，立即退出 gdb 调试器本身
