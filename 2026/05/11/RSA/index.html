欧拉函数 φ(n)

φ(n) 表示 “小于等于 n 且与 n 互质的正整数的个数”

  • 如果 n 是质数 p，那么 φ(p) = p - 1（因为 1,2,…,p-1 都和 p 互质）

  • 如果 n = p * q，且 p、q 都是质数，那么 φ(n) = (p - 1) * (q - 1)

  • 例如 p = 5, q = 11，则 n = 55，φ(55) = 4 * 10 = 40
